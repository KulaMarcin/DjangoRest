# DjangoRest - rest service using django rest framework with:

- Serializers
- APIView
- GenericView 
- Django admin panel
- BasicAuthentication
- SessionAuthentication
- TokenAuthentication
- View set 
- Routers
- Generic view set
- Model view set


# Endpoints: 

## car - APIView:
- http://localhost:8000/car
- http://localhost:8000/detail/2 

## person - Generic view:
- http://localhost:8000/person 
- http://localhost:8000/person/2 

## employee - View set:
- http://localhost:8000/viewset/employee/ 
- http://localhost:8000/viewset/employee/2/ 

## game - Generic view set:
- http://localhost:8000/viewset/game/1 
- http://localhost:8000/viewset/game/ 

## student - Model view set:
- http://localhost:8000/viewset/student/1
